# Access4All — Nonprofit Website

Production website for Access4All-SA, a registered South African nonprofit organisation focused on youth empowerment and support for people living with disabilities.

**Live site: [access4all-sa.co.za](https://access4all-sa.co.za)**

---

## Overview

Built and delivered as sole developer — from initial requirements gathering through to live deployment. This was a real client project with a real public audience, not a portfolio exercise.

---

## Pages & Features

- **Home** — organisation mission, featured programmes, call to action
- **About** — background, team, and organisational values
- **Programmes** — detailed breakdown of active initiatives
- **Donations** — donation information and links
- **Volunteering** — how to get involved
- **Contact** — contact form and organisation details

---

## Tech Stack

- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- Responsive design — mobile and desktop

---

## Project Delivery

| Phase | Details |
|---|---|
| Requirements | Gathered directly from client stakeholders |
| Design | Wireframes and layout agreed with client before build |
| Development | Built solo over approximately 3 months |
| Testing | Cross-browser and mobile testing before handoff |
| Deployment | Fully live with domain and hosting configured |
